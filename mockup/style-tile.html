Profile + About

Variable phosphor

The Matrix code rain has varied weight, alpha, and an occasional cyan-blue hue shift. Applying that signal to every glyph would ruin body legibility, so we reserve it for code-like surfaces: timestamps, citations, IPs, source domains, and the like. Body prose stays uniform.

  • Defends: Reference fidelity to the code rain texture while preserving WCAG body-text contrast on long-form pages.
  • How it works: Code-like spans get class .phosphor-vary. Inside, individual character spans (or words) carry .dim, .bright, .blue modifiers chosen pseudo-randomly at server-render time so output is deterministic.
  • Where it lives in production: .profile__byline, .citations__item, .events__date, .status-bar__value, IP/email mono spans.
